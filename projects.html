Deauther Project

A creative hardware project focused on showcasing the capabilities of Wi-Fi deauthentication using the ESP32 microcontroller. This project is a hands-on exploration of hardware and programming integration.

Components Used:

  • Jumper wires for connections
  • ESP32 microcontroller
  • Breadboard for prototyping
  • Old 0.96" OLED display for visual feedback

    • Features:

  • Ability to scan and deauthenticate devices on Wi-Fi networks (for educational and testing purposes only)
  • User-friendly interface with real-time feedback displayed on the OLED screen
  • Compact and modular design for easy handling
